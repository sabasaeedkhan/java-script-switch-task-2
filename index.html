TASK

Create a grade calculator

  • using a switch
  • Take percentage of marks as input from user
  • And assign him with grades in accordance with his percentage
  • If percentage of marks is >= "90" assign the user with Grade A
  • If percentage of marks is >= "80" assign the user with Grade B
  • If percentage of marks is >= "70" assign the user with Grade C
  • If percentage of marks is >= "60" assign the user with Grade D
  • If percentage of marks is >= "50" assign the user with Grade E
  • If percentage of marks is < "50" assign the user with Grade F (FAIL)

STEPS

  • Create a function "calculateGrade"
  • use a switch with universal truthiness
  • Apply the above 6 percentage situations as cases
  • To run the funtion , take percentage of marks as input and make a variable " marks_percentage"
  • use "parsefloat" function to display the grade
  • At the end , console the user's grade
